## 🎨 统一设计系统

### 颜色系统
```typescript
// 紫粉渐变 (主题色)
colors.gradients.purple = 'from-purple-400 to-pink-500'
colors.gradients.pink = 'from-pink-400 to-rose-500'
colors.gradients.blue = 'from-blue-400 to-cyan-500'
colors.gradients.green = 'from-green-400 to-emerald-500'
colors.gradients.orange = 'from-orange-400 to-amber-500'
```

### 动画系统
```typescript
// 淡入动画
animations.fadeIn = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.5 }
}

// 交错入场动画
animations.listItemEntrance(index) = {
  initial: { opacity: 0, y: 20 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.3, delay: index * 0.05 }
}
```

### 圆角和阴影
- 圆角: `rounded-xl` (12px) 统一
- 阴影: `shadow-lg` → `hover:shadow-xl`
- 间距: `gap-6` (24px) 标准间距

### 响应式断点
- 移动端: `< 768px` - 1列布局
- 平板端: `768px - 1024px` - 2列布局
- 桌面端: `> 1024px` - 3-4列布局
