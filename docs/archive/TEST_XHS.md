# 小红书功能测试指南

## 功能说明

已成功集成小红书搜索功能，支持以下两种搜索方式：

### 1. 关键词搜索
- 搜索小红书笔记
- 示例关键词：穿搭、美食、护肤、旅游

### 2. 对标账号搜索
- 根据用户ID获取该用户的所有笔记
- 用户ID格式：67e0d1560000000007001a39
- 获取方式：打开小红书用户主页，从URL中复制用户ID

## 测试步骤

1. **选择平台**
   - 点击"小红书"按钮（红色）

2. **选择搜索类型**
   - "笔记搜索"：搜索关键词相关的笔记
   - "对标账号"：搜索特定用户的笔记

3. **输入搜索内容**
   - 笔记搜索：输入关键词（例如：穿搭）
   - 对标账号：输入用户ID（例如：67e0d1560000000007001a39）

4. **查看结果**
   - 文章列表会显示：
     - 作者头像和昵称
     - 点赞数、评论数
     - 配图数量
     - 小红书标识

5. **创作文章**
   - 点击"基于此文章创作"按钮
   - 进入创作流程，使用图生图功能

## 注意事项

1. 需要配置环境变量 `DAJIALA_API_KEY`
2. 小红书不提供阅读量数据（只显示点赞和评论）
3. 对标账号需要输入正确的用户ID格式

## API限制

- 关键词搜索：每页返回约20条结果
- 用户笔记：支持翻页，可获取用户所有笔记
- API调用消耗余额，请注意配额
