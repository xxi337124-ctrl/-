## 📋 必需的 API Keys

部署前，请准备好以下 API 密钥：

| API | 用途 | 获取地址 | 必需 |
|-----|------|----------|------|
| OpenRouter | Gemini 2.5 Pro（文案生成） | [openrouter.ai](https://openrouter.ai) | ✅ 必需 |
| SiliconFlow | 文生图功能 | [siliconflow.cn](https://cloud.siliconflow.cn) | ✅ 必需 |
| Doubao | SeeDream 图生图 | [doubao.com](https://console.volcengine.com/ark) | ✅ 必需 |
| 小红书 Cookie | 小红书搜索 | 浏览器 F12 获取 | ⚠️ 可选 |
