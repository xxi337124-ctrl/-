## ⚠️ 重要提醒

### 部署时注意事项

1. **不要跳过步骤**
   - 每一步都很重要，特别是环境变量和数据库配置

2. **先创建数据库再部署**
   - 必须先在 Vercel 创建 Postgres 数据库
   - 然后才能开始部署

3. **仔细检查 API Keys**
   - 确保没有多余的空格
   - 确保密钥是有效的

4. **构建命令必须正确**
   - Build Command: `prisma generate && prisma migrate deploy && next build`
   - 这一步很关键，不要遗漏
